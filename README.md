# newsletter

运行接口：

在api_server目录下

```
npx nodemon .\app.js
```

运行项目：

在时文时讯项目目录下

（1）通过express插件，Ctrl+Shift+P运行express插件

（2）通过live server 插件，鼠标右键open with live server
